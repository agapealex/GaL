# GaL
Sa se realizeze o aplicatie Web menita a oferi suport pentru invatarea componentelor principale ale unui joc (clasic sau electronic), la alegere â€“ a se studia si Outline of games. Se vor lua in consideratie scopul, categoria, regulile, provocarile si manierele de interactiune cu jocul respectiv. Sistemul va monitoriza progresul inregistrat de fiecare utilizator in procesul de invatare, eventual structurata pe niveluri de dificultate. Se va genera, de asemenea, un clasament al celor mai de succes utilizatori, pentru fiecare categorie de jocuri, disponibil si ca flux de date RSS. Bonus: realizarea unui mini-joc de testare a cunostintelor dobandite pe baza sistemului Web implementat.
